git命令有很多，在此记录一下常用的命令。

查看远程分支 ​

git branch -a

创建并且换dev分支 ​

git checkout -b dev

将master合并的dev分支 ​

git merge master
git push origin dev
1
2

其他操作 ​

日志详细
git log

日志版本
git log --pretty=oneline

回退到上一个版本
git reset --hard HEAD^

到指定版本
git reset --hard 版本号

分支指定合并到某个版本
git cherry--pick 版本号

把其他<name>分支合并到当前分支
git merge <name>
git push origin <点前分支>

刷新新的分支信息
git remote update origin -p
